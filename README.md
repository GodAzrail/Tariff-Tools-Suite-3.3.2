## 🧑‍💻 Установка

### Вариант 1: Установка как браузерное расширение (Chrome / Edge)

1. Откройте браузер.
2. Перейдите по адресу:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. Включите **Режим разработчика** (Developer mode).
4. Нажмите **Загрузить распакованное расширение** (Load unpacked).
5. Выберите папку проекта `Tariff-Tools-Suite`.

После установки расширение автоматически активируется на поддерживаемых страницах внутреннего конфигуратора.

## 📦 О проекте

**Tariff Tools Suite** — это профессиональное браузерное расширение (Manifest V3) для внутренней системы конфигурации тарифов доставки на платформе **DNS-Shop** (домен `mow-delivery-calculator.dns-shop.ru`).

Расширение предназначено для сотрудников, работающих с тарифами пунктов выдачи заказов (ПВЗ) и доставки. Оно значительно ускоряет выполнение рутинных задач:

- экспорт детальных данных тарифов;
- массовое создание, обновление и переименование тарифов из Excel;
- гибкую кастомизацию визуального представления карточек тарифов;
- ускоренное заполнение цен в диалогах;
- улучшение удобства работы со служебными модальными окнами.

Проект инжектит разные наборы content scripts в зависимости от открытого раздела (`/configurator/tariffs` или `/configurator/del-orgs`) и использует `localStorage` + SheetJS для обмена данными и работы с файлами Excel.

- 📤 **Экспорт тарифов** с полным сбором данных из карточек и модальных окон

- ⚡ **Автозаполнение ценовых полей** внутри диалогов с поддержкой bulk-операций

## 📂 Описание файлов

### Служебные и конфигурационные файлы

#### `manifest.json`
Главный манифест расширения (Manifest V3).  
Определяет:
- название, версию и иконки;
- permissions: `activeTab`, `scripting`, `storage`;
- `host_permissions` на домен `https://mow-delivery-calculator.dns-shop.ru/*`;
- popup;
- точные правила инъекции content scripts (разные наборы скриптов для `/tariffs*` и `/del-orgs*`, `run_at: document_idle`).

#### `popup.html` и `popup.js`
Всплывающее окно расширения. Отображает статус подключения и проверяет, открыта ли поддерживаемая страница конфигуратора.

#### `plugin-core.js`
**Центральный utility-модуль** проекта. Предоставляет:
- определение текущего роута;
- обёртки и helpers для `localStorage`;
- DOM-утилиты и безопасные селекторы;
- управление сайдбарами (логи, прогресс, конфигурация);
- расчёт responsive-высоты окон;
- диагностику состояния загруженных модулей.

Используется практически всеми остальными скриптами.

#### `content.css`
Глобальные стили, инжектируемые на поддерживаемые страницы. Используются для оформления сайдбаров, панелей управления, кнопок и других элементов интерфейса расширения.

#### `xlsx.full.min.js`
Полноценная минифицированная библиотека **SheetJS**. Отвечает за чтение и генерацию Excel-файлов во всех модулях массовых операций.

#### `.gitattributes`
Служебный файл Git с настройками обработки файлов репозитория.

#### `icons/icon-48.png`, `icons/icon-128.png`
Иконки расширения для панели браузера и страницы управления расширениями.

### Инструменты для работы с тарифами

#### `tariff-tools-content.js`
**Основной content script** для раздела тарифов.  
Отвечает за:
- добавление UI экспорта и кнопок управления;
- инициализацию и управление сайдбаром состояния/логов;
- сохранение состояния в `localStorage`;
- координацию процесса экспорта и взаимодействия с другими модулями.

#### `tariff-page-collector.js`
Специализированный сборщик данных.  
Активируется на страницах детального просмотра тарифа (`/tariffs/*/*`).  
Ждёт полной загрузки модального окна, извлекает все данные тарифа (название, цены, услуги, интервалы и т.д.) и сохраняет их в `localStorage` под ключом `collected_tariff_data`.

#### `tariff-creator.js`
Модуль **массового создания новых тарифов** из Excel.  
Загружает файл, показывает конфигурационный и прогресс-сайдбар, ведёт подробный лог и пошагово создаёт тарифы через интерфейс сайта.

#### `pvz-tariff-creator.js`
Специализированная версия логики создания тарифов, адаптированная под особенности работы с ПВЗ (пунктами выдачи заказов).

#### `tariff-updater.js`
Модуль **массового обновления существующих тарифов** на основе данных из Excel-файла.  
Считывает файл, управляет конфигурацией импорта, логирует каждый шаг и применяет изменения к уже существующим тарифам.

#### `tariff-renamer.js`
Модуль **массового переименования тарифов**.  
Генерирует шаблон для выгрузки текущих названий, читает Excel с колонками «старое имя → новое имя» и автоматически выполняет переименование с отображением прогресса и логов.

#### `tariff-customizer.js`
Мощный модуль **кастомизации визуального представления** страницы тарифов.  
Возможности:
- добавление тарифов в избранное и VIP/супер-категории;
- закрепление карточек;
- пользовательский порядок и drag-and-drop;
- сохранение и восстановление раскладки из `localStorage`;
- адаптивная переразметка колонок.

#### `tariff-tools-patch.js`
**Устаревший legacy-модуль** (в настоящее время не подключается в `manifest.json`).  
Оставлен в репозитории для истории. Выводит сообщение о том, что отдельный патч больше не требуется (дедупликация логов перенесена в основной модуль).

### UI-утилиты и улучшение диалогов

#### `autofill-content.js`
Инструмент **быстрого заполнения цен** в диалогах тарифов.  
Находит таблицу с ценовыми полями, встраивает компактную панель управления, позволяет:
- вставлять значения пачкой;
- применять значения ко всем строкам;
- копировать результат в Excel-совместимом табличном виде.

#### `resize-core.js`
**Базовый модуль** системы ресайза диалогов.  
Предоставляет общие функции расчёта высоты, позиционирования окна и наблюдения за появлением модальных окон. На странице `/configurator/tariffs` изменение размеров специально блокируется (основной фокус — страница `del-orgs`).

#### `resize-dialog.js`
Улучшает диалог `select-volume-weight-dlg` (создание наборов МГХ/сетки). Увеличивает окно, улучшает прокрутку и компоновку содержимого.

#### `resize-intervals-dialog.js`
Улучшает диалог `select-intervals-dlg` на странице `del-orgs`. Делает окно выше и удобнее для работы с большим объёмом данных.

#### `resize-branches-dialog.js`
Улучшает диалог `select-branches-dlg` на странице `del-orgs`. Увеличивает окно, перераспределяет внутренние блоки и делает список филиалов удобнее для просмотра.

#### `resize-schedule-dialog.js`
Улучшает диалог `select-schedule-dlg`. Работает с таблицей недельного расписания, наблюдает за состоянием диалога и улучшает размер и поведение для удобного редактирования графика.

## 🔗 Взаимодействие модулей

1. `manifest.json` определяет, какие скрипты и когда инжектить.
2. `popup.html` + `popup.js` дают быстрый статус по активной вкладке.
3. `plugin-core.js` предоставляет общие утилиты всем модулям.
4. `tariff-tools-content.js` + `tariff-page-collector.js` реализуют экспорт.
5. `tariff-creator.js`, `tariff-updater.js`, `tariff-renamer.js` и `pvz-tariff-creator.js` отвечают за Excel-сценарии.
6. `tariff-customizer.js` управляет визуальной организацией страницы.
7. `autofill-content.js` и группа `resize-*` скриптов улучшают UX диалогов.
8. `xlsx.full.min.js` используется всеми модулями, работающими с Excel.

## ⚠️ Известные особенности и ограничения

- Расширение работает **только** на указанном внутреннем домене DNS-Shop.
- Массовые операции (create/update/rename) используют симуляцию действий пользователя — при значительных изменениях DOM целевого сайта может потребоваться обновление селекторов.
- Состояние и логи сохраняются в `localStorage` браузера (ограниченный объём).
- Для стабильной работы bulk-операций рекомендуется не закрывать вкладку и не выполнять параллельно другие действия.
- Файл `tariff-tools-patch.js` является legacy и не влияет на текущую работу.
